## What is this?

The DNS Incident Timer is a fun desk gadget that displays "Days Since Last DNS Incident" on an RGB LED matrix. When DNS inevitably causes problems, press the big red button to reset the counter and play a shameful audio clip.

<CardGrid stagger>
	<Card title="LED Matrix Display" icon="star">
		Bright, visible counter showing days (or hours, or minutes) since the last incident.
	</Card>
	<Card title="Physical Reset Button" icon="warning">
		Satisfying tactile button to reset the counter when DNS strikes again.
	</Card>
	<Card title="Audio Alerts" icon="setting">
		Plays a customizable audio clip when the counter resets.
	</Card>
	<Card title="Web Interface" icon="laptop">
		Optional web UI for remote monitoring and reset.
	</Card>
</CardGrid>
